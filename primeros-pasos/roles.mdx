En NADIA hay dos niveles de acceso: **usuario estándar** y **administrador**.

## ¿Qué ve cada rol?

| Sección | Usuario estándar | Administrador |
|---------|:---------------:|:-------------:|
| Panel general | Sus propias citas | Vista completa de la clínica |
| Citas | ✅ | ✅ |
| Pacientes | ✅ | ✅ |
| Bonos | ✅ | ✅ |
| Pools | ✅ | ✅ |
| Actividades dirigidas | ✅ | ✅ |
| Historial de consumos | ✅ | ✅ |
| Asistente IA | ✅ | ✅ |
| WhatsApp | ✅ | ✅ |
| Mi perfil | ✅ | ✅ |
| Estadísticas | ❌ | ✅ |
| Facturación | ❌ | ✅ |
| Empresas | ❌ | ✅ |
| Profesionales | ❌ | ✅ |
| Ajustes | ❌ | ✅ |

<Note>
Si una sección no aparece en tu menú, es porque tu cuenta no tiene acceso a ella. Contacta con el administrador si crees que debería ser visible para ti.
</Note>

## Roles clínicos

Además del nivel de acceso al sistema, cada profesional tiene un **rol clínico** (fisioterapeuta, nutricionista, entrenador, etc.). Este rol determina cómo aparece el profesional en el calendario y en las fichas, pero **no afecta a qué secciones puede ver en el dashboard**.

El acceso al dashboard lo determina únicamente si la cuenta es de usuario estándar o administrador.
